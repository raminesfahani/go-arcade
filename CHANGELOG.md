## [Unreleased]

### Added

- Initial project scaffold and documentation
- Minimal 2D arcade game demo using Ebiten
- Player movement with arrow keys and WASD
- Basic shooting mechanics with spacebar
- Procedurally generated sprite rendering
- Audio module scaffolding for sound implementation
- Internal UI utilities for game rendering
- GitHub Actions CI workflow for automated testing
- Comprehensive README with quick start guide
- Contributing guidelines and code of conduct
- MIT License
